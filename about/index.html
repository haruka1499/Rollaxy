NOVORA GAME について

NOVORA GAME とは

NOVORA GAME は、ブラウザで無料で遊べるゲームを公開している個人運営のサイトです。 インストールやアカウント登録は不要で、スマートフォン・PC どちらからでもすぐに遊べます。

ゲームについて

現在公開中のゲームは Games ページ からご覧いただけます。 今後も新しいゲームを順次追加していく予定です。

公開中のゲームはすべて HTML / CSS / JavaScript のみで動作しており、 外部サーバーへの通信は行いません。スコアなどのデータはブラウザのローカルストレージに保存されます。
